SafeGen Extension Installation and Setup Guide

Step 1: Download and Install the SafeGen Extension

To begin, download the SafeGen extension ZIP file from the link below:

Download SafeGen Extension

  1. Unzip the downloaded ZIP file to a folder on your computer.
  2. Open Chrome and go to chrome://extensions by typing it into the address bar and hitting Enter.
  3. In the top-right corner, toggle the switch to enable Developer mode.
  4. Click the Load unpacked button.
  5. Navigate to the folder where you unzipped the SafeGen extension and select the folder.
  6. The SafeGen extension will now appear in your list of extensions.

Step 2: Generate a GitHub Personal Access Token

Next, you'll need to generate a Personal Access Token from GitHub to allow the SafeGen extension to work with your gists:

  1. Login to your GitHub account.
  2. Go to this link: Generate GitHub Token
  3. Click on the Account Permissions option.
  4. Allow read and write access for gists.
  5. Click Generate token.
  6. Copy the generated token.

Step 3: Configure the SafeGen Extension in Chrome

Now that the extension is installed and you have a GitHub token, follow these steps:

  1. In Chrome, go to the Extensions menu and pin the SafeGen extension for easy access.
  2. Click on the SafeGen extension icon in the toolbar to open its home page.
  3. Paste the GitHub token you generated earlier into the input field on the home page.
  4. Click Save.

Step 4: Setting Up SafeGen on Microsoft Edge

If you want to use SafeGen on Microsoft Edge as well, follow these steps:

  1. Open Edge and go to edge://extensions in the address bar.
  2. Enable Developer mode by toggling the switch in the top-right corner.
  3. Click Load unpacked and select the same folder where you unzipped the SafeGen extension.
  4. Once the extension is loaded, click the SafeGen icon in Edge's toolbar to open it.
  5. Paste the same GitHub token you used in Chrome.
  6. Click Save to sync your extension in Edge.
  7. Now you can use SafeGen across both browsers with the same token!
