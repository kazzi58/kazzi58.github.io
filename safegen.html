SafeGen Extension Installation and Setup Guide

Step 1: Generate a GitHub Personal Access Token

To allow the SafeGen extension to work with your GitHub account, you'll need to generate a Personal Access Token:

  1. Go to your GitHub account and open the Developer Settings:
    • Click on your profile picture in the top-right corner.
    • Select Settings.
    • Scroll down and click Developer settings.
    • In the left-hand menu, click Personal access tokens.
  2. Generate a new token:
    • Click Generate new token.
    • Give it a name (e.g., "SafeGen Access").
    • Under Select scopes, check the box for gist to allow read and write access.
    • Scroll down and click Generate token.
    • Copy the generated token.

Step 2: Replace the Token in the Extension

Before loading the extension, you'll need to replace the placeholder token in the code:

  1. Extract the ZIP file of the extension to a folder on your computer.
  2. Navigate to src/gistOperations.js.
  3. Open gistOperations.js in any text editor (e.g., Notepad, VSCode).
  4. Find the line where the token is defined:
    5. const GITHUB_TOKEN = "your-token-here";
  5. Replace "your-token-here" with the token you copied from GitHub.
  6. Save the file.

Step 3: Enable Developer Mode in Chrome

To load the extension, you'll need to enable Developer mode in Chrome:

  1. Open Chrome and go to the Extensions page by typing chrome://extensions/ in the address bar and hitting Enter.
  2. In the top-right corner, toggle the switch to enable Developer mode.

Step 4: Load the Extension

Now you can load the SafeGen extension into Chrome:

  1. On the Extensions page (with Developer mode enabled), click Load unpacked.
  2. Navigate to the folder where you extracted the extension (and already replaced the token).
  3. Select the folder.
  4. The SafeGen extension will now appear in your list of extensions.

If you encounter any issues during the setup, feel free to contact us for assistance.
